## What is TinaDocs?

TinaDocs is a documentation starter built with TinaCMS and NextJS. It's the most efficient way to
build your own documentation site.

## How to use it

## MDX Components Included in this starter

### Callouts

Callouts are used to highlight specific blocks of content within your documentation.

The types of callouts are:

* Information
* Warning
* Error
* Success
* Idea
* Locked
